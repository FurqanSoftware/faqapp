# Faqapp

![](assets/logo_128.png)

Faqapp is a simple Go Markdown based knowledge base app by Furqan Software.

## Example

At [Furqan Software](https://furqansoftware.com/), Faqapp powers [Toph Help](https://help.toph.co/).
